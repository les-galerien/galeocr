**To launch the project**

```
virtualenv -p python3 .venv
```

```
source .venv/bin/activate
```

```
pip install -r requirements.txt
```

**OR**
Copy paste the commands in the `requirements-update.txt`

**THEN**
To escape the venv, use `deactivate` command
